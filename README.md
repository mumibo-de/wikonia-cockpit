# 🌐 Wikonia Cockpit

Das zentrale Verwaltungs-Backend für das Wikonia-Wiki-Netzwerk.  
Ermöglicht Kundenverwaltung, Wiki-Provisionierung, Konfiguration und mehr – entwickelt mit Laravel, Filament und viel Liebe zur Struktur™.
